# 🎯 DARTS SCORER — ハード&ソフト二刀流スコアラー

スティールダーツ(ハード)とソフトダーツ、**2つの文化をモード切替ひとつで丸ごと行き来できる**スコアリングアプリ。
ハードはパブの木目とノック音、501レグ戦、チェックアウト定石提案、180コール。ソフトはアーケードの電子音、Rt/80%スタッツ、メドレー、プロ試験練習。プレイヤーは共通プロフィールで、両文化の成績を一人の中に積み上げます。

> React + Vite 製。エンジンは純関数、UIとロジックを分離し、vitestで挙動を固定。

## ✨ 主な機能

- **01 / クリケット / カウントアップ / メドレーマッチ**(チーム戦・コーク・シャッフルスタート対応)
- **モードで文化が切り替わる**: ゲームの品揃え、ルール既定値、マッチ形式(ハード=501レグ連戦、ソフト=メドレー)、練習メニュー、スタッツ体系、テーマ(木目パブ調⇔アーケード調)、効果音(木のノック⇔電子音)の7層が連動
- **チェックアウト提案**: 2〜170の定石チャート162エントリ。全エントリを機械検証し、**「提案通りに投げ続ける限り提案が変わらない」継続性をテストで保証**。持ちダーツ数・ファットブル設定にも追従
- **ROBO RIVAL**: 2次元正規分布の着弾シミュレーションAI(Lv1〜9)。σはモンテカルロで実レーティングに校正済み。アレンジ(残し計算)も実装
- **デュアルレーティング**: DARTSLIVE方式(Rt・80%スタッツ)とPHOENIX方式(Rt1-30・100%スタッツ)を公式換算表ベースで併記
- **ハードスタッツ文化**: 3ダーツAVG、ファースト9、チェックアウト率、ハイチェックアウト、TON/180カウント、フルスクリーン180演出+歓声SE
- **練習モード**(モード別): ハード=Around the Clock / Bob's 27 / 121、ソフト=クリカン / HALF-IT / SHOOT OUT(プロ試験種目、ターゲット以外のボード暗転つき)。自己ベストを永続化
- **刺さるダーツUI**: タップ座標にフライト付きダーツが刺さる。スティールは打撃合成音(steel thock)
- **縦・横画面対応**: 横長ではスコア/ボードの2カラムへ自動再構成

## 🏗 アーキテクチャ

```
src/
├── main.jsx              # エントリ(storageアダプタ → App)
├── App.jsx               # 画面遷移・設定・プロフィール永続化
├── theme.js              # 2系統テーマ(ライブバインディングでアプリ全体を切替)
├── constants.js          # ゲーム定義・カテゴリ・各種シーケンス
├── sound.js              # Web Audio合成SE(電子音系 / 木・スティール系)
├── profiles.js           # プロフィール・レーティング(DL/PX)・スタッツ枝
├── storage.js            # window.storage互換のlocalStorageアダプタ
├── engine/               # 純関数ゲームエンジン(UI非依存・全てテスト対象)
│   ├── game.js           #   newGame / applyDart / endTurn / アワード判定
│   ├── checkout.js       #   定石チャートとルート解決
│   ├── board.js          #   ボード幾何・セグメント変換
│   └── robo.js           #   着弾シミュレーションAI
├── ui/                   # 汎用コンポーネント(キット・ボード・テンキー)
└── screens/              # 画面(home / flow / game / players / match)
```

設計方針:

- **エンジンは純関数**(`state in → state out`、deep copyで不変)。undoは状態スナップショットで実現し、UIから完全に独立
- **テーマはESMライブバインディング**: `theme.js`のパレット/フォントをAppがレンダリング前に差し替え、全コンポーネントが一括で追従
- **ストレージは互換アダプタ**: Claudeアーティファクト由来の`window.storage` APIをlocalStorageで再現し、同一コードがスタンドアロン/PWAで動作

## 🧪 テスト

```bash
npm test
```

vitestで17ケース。特に重要なもの:

- チェックアウトチャートの**全数検証**(合計一致・最終投がダブル/ブル・3投以内・ボギーナンバー)
- **継続性シミュレーション**: 全162エントリで「提案通りに投げたら提案が最後まで一度も変わらない」ことを機械的に保証
- 練習ゲーム6種のルール(Bob's 27の減点/脱落、121の昇降、SHOOT OUTの倍率とブル復活、HALF-ITの半減連鎖 など)
- 01スタッツ(トン帯の排他カウント、ファースト9、チェックアウト点)とマッチ構成のモード分岐

## 📦 公開手順

**GitHub**

```bash
git init && git add -A && git commit -m "feat: darts scorer v1"
git remote add origin git@github.com:<you>/darts-scorer.git
git push -u origin main
```

**Vercel**(推奨・無料枠で十分)

1. vercel.com → Add New Project → GitHubリポジトリをインポート
2. Framework Preset: **Vite**(自動検出)。Build Command `npm run build` / Output `dist` のままでOK
3. Deploy。以後は`main`へのpushで自動デプロイ

GitHub Pagesの場合は `vite.config.js` に `base: "/darts-scorer/"` を追加し、`dist`を`gh-pages`ブランチへ。
